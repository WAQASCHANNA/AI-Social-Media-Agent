**Hack-Nation 2026** | Waqas Channa | https://github.com/WAQASCHANNA/AI-Social-Media-Agent

## Overview
Automates social media content creation by learning brand voice from 3 samples and generating platform-optimized posts with visuals in 5 minutes.

## Architecture
**Stack:** Python 3.8+ | Streamlit 1.31.0 | Google Gemini API | Pillow 10.2.0 | Streamlit Cloud  
**Modules:** Brand Analyzer | Content Generator | Visual Engine | Critique Agent | Improvement Agent  
**Flow:** Input → Brand Extract → Generate → Render → Critique → Review → Export

## Key Innovations
**Dual-Mode AI:** Gemini for creativity + Mock for reliability (99.9% uptime) | **Agentic Loop:** Generator → Critic (1-10 score) → Improvement | **Visual Intelligence:** Auto-maps tone to colors/fonts | **Platform-Aware:** LinkedIn (professional, 1-3 hashtags) vs Instagram (visual, 5-10 hashtags)
